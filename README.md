## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/venkatesh-r/portfolio.git
   cd portfolio
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
4. **Open in browser**
   - Visit `http://localhost:3000/`
