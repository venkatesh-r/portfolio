## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/venkatesh-r/dev-tinder-ui.git
   cd dev-tinder-ui
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run the development server**
   ```bash
   npm run dev
   ```
4. **Open in browser**
   - Visit `http://localhost:3000/`
